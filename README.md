
### Good Job Calender

당신의 안정적인 커리어 첫 이륙을 위해서 더 이상 걱정 없는 취준 알람 캘린더, Good Job Calender!

### 🚀 기능

#### 😃 간단하고 안전한 회원가입

- 카카오를 통한 소셜 로그인으로 간단하고 안전하게 회원가입 할 수 있어요!
